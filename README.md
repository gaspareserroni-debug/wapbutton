# WhatsDial (WA Button Overlay)

Mostra un pulsante **WA** sopra l'app Telefono: toccandolo apre WhatsApp per il numero visualizzato.

## Build su GitHub Actions
- Carica questa cartella su un repo.
- Actions → Run workflow → scarica l'artefatto *app-debug.apk*.

## Permessi da abilitare sul telefono
- Overlay su altre app
- Servizio di Accessibilità "WhatsDial Overlay"
